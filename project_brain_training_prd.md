# 脑力训练 Web 游戏项目策划书 (Project Brain Training PRD)

> **文档状态**: 草稿 (Draft)  
> **最后更新**: 2026-02-06  
> **目标读者**: 开发团队、产品负责人  

## 1. 产品概述 (Product Overview)
**项目名称**: 脑力心流 (Brain Flow) - 暂定  
**核心目标**: 打造一款基于 Web 的商业化脑力训练平台，通过极简的视觉体验和科学的 N-Back 训练机制，帮助用户提升工作记忆与专注力。  
**目标用户**: 追求自我提升的学生、职场人士，以及喜爱益智类游戏的休闲玩家。  
**风险控制 (版权规避)**: score = (accuracy × nLevel × totalRounds) / 10
- **视觉层面**: 摒弃原作的“教授头像”、“写实图标”等元素，采用类似《2048》的**极简几何/数字风格**。
- **玩法层面**: 核心算法基于通用的神经科学范式 (N-Back)，而非特定游戏的专利玩法，通过原创的 UI 交互进行包装。

## 2. 核心玩法与差异化设计 (Core Gameplay & Differentiation)
**整体风格**: **极简禅意 (Minimalist Zen)**。强调“心流”体验，无干扰的纯净界面。

### 2.1 基础游戏模式 (MVP 阶段)
1.  **数字心流 (Numeric Flow)** - *对标“魔鬼计算”*
    -   **玩法**: 屏幕显示简单的数学题（如 3+5），玩家需输入 N 回合前的答案。
    -   **表现**: 大字号、柔和配色。数字淡入淡出，错误时屏幕轻微震动（视觉）。
2.  **空间心流 (Spatial Flow)** - *对标“魔鬼老鼠”/“魔鬼形状”*
    -   **玩法**: 在 3x3 或 4x4 网格中，光点或色块按顺序移动/闪烁。玩家需指出 N 回合前的位置。
    -   **表现**: 类似 Launchpad 的按键灯光效果，扁平化色块。
3.  **思维心流 (Verbal Flow)** - *对标“魔鬼朗读”*
    -   **玩法**: 快速阅读短语或成语，记住并回填高亮的核心词汇。

### 2.2 难度动态调节
-   **动态 N-Back**: 系统根据玩家正确率实时调整 N 值（2-Back, 3-Back...），保持心流体验（不过难也不过简）。

## 3. 经济系统与商业化 (Economy & Monetization)
**核心循环**: 获取积分 -> 消耗积分训练 -> 提升能力/排名 -> 获取更多奖励

### 3.1 资源定义
-   **体力 (Energy)**: 限制每日游玩次数（防沉迷/拉长生命周期）。例如每日恢复 5 点体力。
-   **积分 (Brain Points)**: 游戏内的核心货币。

### 3.2 积分获取 (Sources)
1.  **每日打卡 (Daily Check-in)**: 连续打卡奖励递增。
2.  **充值 (IAP)**: 直接购买积分/体力包（商业化核心）。
3.  **挑战奖励 (Challenges)**: 完成“每日 3 次 3-Back 训练”等任务。
4.  **成绩结算**: 每次训练根据表现给予少量积分。

### 3.3 积分消耗 (Sinks)
1.  **补充体力**: 超过每日免费次数后，需消耗积分继续训练。
2.  **解锁主题**: 购买不同的配色皮肤（如“深邃夜空”、“樱花粉”、“极客绿”）。
3.  **复活/容错**: 在挑战模式中消耗积分抵消一次错误（可选项，需慎用以保公平）。

## 4. 社交与排行榜系统 (Social & Leaderboards)
### 4.1 排行榜 (Leaderboards)
-   **维度**:
    -   **极限榜**: 记录玩家达到的最高 N-Back 等级（如 "8-Back 大神"）。
    -   **速度榜**: 在固定 N 值下的答题速度/准确率综合分。
-   **周期**:
    -   **周榜**: 每周重置，激发活跃度。
    -   **总榜**: 记录历史最高荣誉。

### 4.2 个人档案 (Profile)
-   **脑力雷达图**: 展示 记忆力 (Memory)、计算力 (Math)、专注力 (Focus) 三维数据。
-   **历史曲线**: 展示用户过去 30 天的能力变化趋势。

## 5. 视觉与交互风格 (UI/UX Design)
-   **设计语言**: Flat Design (扁平化)。
-   **配色**: 莫兰迪色系（低饱和度），保护视力，久看不累。
-   **动效**: 丝滑的 CSS3 动画，强调操作的即时反馈。
-   **响应式**: 完美适配 PC 浏览器与手机浏览器（H5）。

## 6. 技术架构概要 (Technical Architecture Overview)
-   **前端**: React + TypeScript + Tailwind CSS (快速构建极简 UI)。
-   **状态管理**: Zustand 或 Redux (管理游戏状态与 N-Back 队列)。
-   **后端 (未来规划)**: Node.js/Next.js API Routes + PostgreSQL (存储用户数据、排行榜)。
-   **部署**: Vercel 或 Netlify (低成本起步)。
